隐私政策

最后更新：2026年2月

🛡️

数据收集声明

AI 校对鸭承诺：我们不收集、读取或存储用户的任何个人信息。

我们的扩展旨在提供高质量的校对和写作辅助服务，同时保护用户隐私。所有操作均在您的设备上本地执行，或通过安全的用户配置 API 执行，不会将个人数据发送到我们的服务器。

✨ 您的数据始终属于您

🔑

API 与模型安全

本扩展使用本地 AI 模型 (WebLLM) 或 Google Gemini API 提供智能文本处理。

  • 您的 API 密钥仅存储在您浏览器的本地存储中
  • API 请求直接从您的设备发送到 Google 服务器 (如使用 Gemini API)
  • 我们不收集、传输或存储您的 API 密钥或模型配置
  • 您可以随时删除或更改您的 API 密钥或模型设置
🔒

我们不收集的信息

为了明确表达我们对隐私的承诺，以下是我们绝不收集的信息：

  • 您的个人身份信息
  • 您的浏览历史或访问的网站
  • 您进行校对或重写的文本内容
  • 您的 IP 地址或地理位置
  • 您的 API 密钥或令牌
  • 除通用诊断信息外的任何设备或系统信息
🔧

浏览器权限说明

本扩展需要以下浏览器权限以提供核心功能:

  • storage: 保存您的设置、API 密钥和语言偏好到本地存储
  • sidePanel: 在浏览器侧边栏中显示处理结果
  • activeTab: 读取您选中的文本内容进行处理
  • contextMenus: 在右键菜单中添加快捷操作

这些权限仅用于扩展的核心功能,不会被用于任何数据收集或跟踪目的。

⏳

数据保留与用户权利

数据保留:

  • 所有数据(设置、API 密钥)仅存储在您的浏览器本地存储中
  • 处理的文本内容不会被保存,处理完成后立即从内存中清除
  • 卸载扩展时,所有本地数据将被自动删除

您的权利:

  • 随时在扩展设置中查看、修改或删除您的配置
  • 随时切换推理引擎(本地模型或在线 API)
  • 随时卸载扩展,所有数据将被完全清除
🌐

第三方服务

如果您选择使用在线 API 模式,您的文本将被发送到您配置的 API 服务提供商(如 Google Gemini)。请注意:

  • 第三方服务的隐私政策由其服务提供商制定
  • Google Gemini API 隐私政策: https://policies.google.com/privacy
  • 我们建议您在使用前仔细阅读相关服务的隐私政策
  • 您可以随时切换到完全本地的 WebGPU 或 WASM 模式以避免数据传输
📬

联系我们

如果您对本隐私政策有任何疑问或建议，请在我们的 GitHub 仓库中提交 Issue。我们承诺尽快回复您的反馈。

政策更新: 我们可能会不定期更新本隐私政策。重大变更将在扩展更新说明中通知用户。
